Privacy Pollicy

The bot will:
- Store basic data about your server (Name and Id)
- Read any message that starts with the configured prefix (? by default)
- Read all data sent by interactions/slash commands (Command Data and User)
- Temporarily access all server and member data
The bot will never:
- Store server or member data (Asside from the Name and Id of the server)
- Store message or interaction/slash command data
